How main.js builds this effect
01 · YA / j9 / QA
N-D Grid + Neighbourhood
Creates a flat Uint32Array grid and precomputes all neighbour offsets for fast spatial lookups.
02 · XA
Box-Muller Sampler
Generates normally-distributed random unit vectors, giving each particle a smooth, organic direction.
03 · ri / Gn / tc
Poisson Disk Sampling
Places points with guaranteed minimum spacing — no clumping, no gaps. Fixed-density or variable via distanceFunction.
04 · nI / e7
Three.js Scene
WebGLRenderer + PerspectiveCamera + Points mesh. Raycaster maps cursor → 3D intersection point each frame.
05 · GSAP Ge
GSAP Animations
onHoverStart/End drives hoverProgress and pushProgress tweens that feed into shader uniforms each frame.
06 · Angular tM
Root App + Router
Bootstraps Angular, manages scroll restoration, and fires analytics on every NavigationEnd event.
